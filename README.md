# Academic Advising System - Frontend

## Technologies Used
- React
- Vite
- JavaScript
- CSS

---

## How to Run the Project

### 1. Clone Repository

```bash
git clone https://github.com/USERNAME/frontend-repo.git
```

### 2. Open Project Folder

```bash
cd frontend-repo
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run Development Server

```bash
npm run dev
```

### 5. Open in Browser

```text
http://localhost:5173
```
